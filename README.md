# CVAT Annotation Portfolio 🎨

Привет! Это мой проект по разметке данных для задач компьютерного зрения.

## 🛠 Инструменты
- **CVAT (Computer Vision Annotation Tool)**
- Формат экспорта: **CVAT for images 1.1**

## 📋 Описание задач
В рамках этого проекта я разметила следующие объекты:
- 🚗 **Автомобили**: Bounding Boxes (прямоугольники).
- 🐶 **Животные**: Bounding Boxes (прямоугольники).
- 👤 **Люди**: Polygons (полигоны/сегментация).

## 🔧 Используемые инструменты разметки
- **Rectangle** (Прямоугольник) - для быстрой разметки автомобилей и животных
- **Polygon** (Полигон) - для точной сегментации людей по контуру

## 📊 Детали проекта
- **Количество изображений**: 3
- **Количество размеченных объектов**: 20+
- **Типы разметки**: 
  - Object Detection (детекция объектов)
  - Segmentation (сегментация)

## 📂 Структура проекта
- `images/` - исходные изображения
- `annotations/` - файл разметки в формате CVAT XML
